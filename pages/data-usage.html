Dataset Overview

This dataset contains architectural floorplan data enriched with performance metrics including energy analysis, daylight simulation, and spatial relationships.

Included Data:

  • IFC building models with geometric data
  • Energy performance analysis results
  • Daylight simulation outputs
  • Spatial adjacency relationships
  • Building metadata and classifications
