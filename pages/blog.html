Performance-data Enriched Floorplan Datasets

September 24, 2025 By Marco Durand, Matea Pinjusic, and Joaquin Broquedis IAAC - Institute for Advanced Architecture of Catalonia

Abstract

The integration of generative AI into architecture is transforming early-stage design processes, particularly by enabling rapid and automated floorplan generation. These tools increase efficiency by exploring large design spaces and producing multiple layout options in a short time. However, current workflows struggle to integrate critical performative analyses such as daylight availability, thermal comfort, and energy demand into the earliest phases of design.

This project develops a real-world architectural dataset enriched with geospatial information (GeoJSON), environmental data, and energy simulation outputs. The dataset provides a foundation for advancing performance-informed generative design, enabling machine learning models and design tools to account for climatic and geographic conditions alongside spatial and geometric qualities of buildings.

Key Research Questions

"How can we create a workflow to enrich existing floorplan databases with performative data?"

Methodology Highlights

  • Swiss Dwellings Dataset: Real Swiss residential buildings with normalized CSV tables and WKT geometry
  • Context Integration: Urban meshes via Cityweft API with climate-specific EPW files
  • Performance Analysis: Ladybug Tools for daylight and solar analyses in Grasshopper
  • Multi-format Export: CSV, XLSX, IFC2x3, and GraphML outputs with stable IDs
  • Graph Networks: Space-adjacency and circulation graphs for ML applications

Dataset Statistics

1,171 Buildings Processed
4,994 Room-level Simulations
10 Climate Zones
782 Building Graphs

Technical Pipeline

  1. CSV Extraction: Parse Swiss Dwellings relational tables into Grasshopper DataTrees
  2. 3D Reconstruction: Convert WKT geometry to parametric 3D models
  3. Context Integration: Match buildings to cities and import urban context via Cityweft
  4. Performance Analysis: Run daylight and solar simulations with consistent grids
  5. IFC Authoring: Create IFC2x3 files with embedded performance data as custom properties
  6. Graph Generation: Extract space-adjacency and circulation networks
  7. Multi-format Export: Synchronized outputs across CSV, XLSX, IFC, and GraphML

Enhanced IFC Viewer with Speckle Integration

October 27, 2025 By Development Team

We've upgraded our IFC viewer to integrate with Speckle for better performance data visualization and interoperability. The new viewer supports the enhanced dataset format with embedded performance metrics.

New Features:

  • Real-time performance data overlay on 3D models
  • Interactive daylight analysis grid visualization
  • Energy flow visualization for thermal performance
  • Speckle stream integration for collaborative workflows
  • Enhanced metadata display with climate and context data

The viewer now displays color-coded performance zones, window orientation data, and space-level energy metrics directly on the 3D model, making it easier to understand the relationship between design and performance.

Interactive Graph Visualizations: 37+ Adjacency Networks

October 25, 2025 By Research Team

Our comprehensive collection of interactive adjacency graphs provides insights into spatial relationships within buildings. Each graph represents the connectivity and circulation patterns of architectural spaces.

Graph Types Available:

  • Space-Adjacency Graphs: Physical connections between rooms and spaces
  • Circulation Networks: Walkable paths and movement flows
  • Performance-Enhanced Networks: Nodes enriched with daylight and energy data

These visualizations support research in wayfinding, egress analysis, program clustering, and machine learning feature extraction for building topology understanding.

Open Dataset for Performance-Informed Design

October 20, 2025 By Research Team

Our enhanced dataset addresses the gap between architectural layouts and environmental performance data. Unlike existing synthetic datasets, our corpus is built from real Swiss residential buildings with comprehensive performance analysis.

Dataset Features:

  • Real-world building geometries from Swiss Dwellings database
  • Performance simulations across 10 climate zones
  • Daylight factor analysis with 1-meter sensor grids
  • Solar radiation and sun hour calculations
  • Energy demand modeling with context-specific parameters
  • Graph network representations for ML applications

This dataset enables researchers to train ML models that consider both spatial configuration and environmental performance, advancing the field of performance-informed generative design.
